# 📊 Sales Data Analysis using Python

## Overview
This project analyzes a furniture sales dataset containing 10,000 transaction records using Python. The project covers data cleaning, data aggregation, data visualization, and business insight generation to support marketing decision-making.

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Project Workflow
1. Data Cleaning
2. Data Aggregation
3. Data Visualization
4. Business Insight

## Key Outputs
- Removed duplicate and missing data.
- Extracted city information from shipping addresses.
- Analyzed sales performance by city and product.
- Built visualizations using bar charts, scatter plots, and box plots.
- Developed business recommendations based on analytical findings.
